## How can I edit this code?

There are several ways to work with this application.



## Use Your Preferred IDE (Local Development)

If you want to work locally using your own IDE, follow the steps below.

### Prerequisites
- Node.js (v18+ recommended)
- npm (comes with Node.js)

Install Node.js using **nvm** (recommended).



### Step-by-Step Setup

```bash
# Step 1: Clone the repository
git clone https://github.com/VADLASAHITYA/stockhom.git

# Step 2: Navigate to the project directory
cd stockhom

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
